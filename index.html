About Me

Hi, I'm Kirenga Isaac — a software and systems developer based in Kampala, Uganda. I specialise in building web applications, system integrations, and networking solutions. This blog started as a solo phone project written entirely in TrebEdit on Android and has since grown into a full-featured platform, now developed in VS Code and powered by Firebase.

With a background spanning Linux system administration, web development, and network engineering, I write practical tutorials and share real-world insights from building production systems in Uganda. My goal is to document what I learn so that other developers — especially those in Africa building with limited resources — can find useful, honest content.

Topics I cover include JavaScript, HTML/CSS, Firebase, Android development, Ubuntu/Linux, dual-boot setups, networking fundamentals, and general software engineering. Whether you are a student, a self-taught developer, or a working professional in tech, there is something here for you.
